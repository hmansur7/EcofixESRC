Downloads
----------
Nodejs
MySql
Python
Github

Steps
-----
1. Clone the Repo
2. Open in VS Code

Frontend
--------
1. Open the frontend folder in an integrated terminal
2. npm install
3. npm start

Backend
-------
1. Open the backend folder in an integrated terminal
2. pip install -r requirements.txt
3. Open MySQLWorkbench
4. Connect to localhost
5. File -> Open SQL Script -> Choose the EcoFixSolutions_Schema.sql
6. CTRL+A the Schema and click the lightining bolt
7. Create a .env in the backend folder
8. Update the password
9. In the integrated terminal run py manage.py makemigrations
10. run py manage.py migrate
11. run py manage.py runserver
